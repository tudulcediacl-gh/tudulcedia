Mejoras a implementar en dashboard actual

1. Mostrar origen cuando el pedido venga desde catálogo Supabase.
2. Agregar búsqueda/filtro rápido por “Supabase”.
3. Mostrar tipo de pedido: galletas, pan o mixto.
4. Mostrar nota de entrega/separación cuando aplique.
5. Mantener WhatsApp dashboard como negocio → cliente.
